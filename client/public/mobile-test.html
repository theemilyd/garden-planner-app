How to test

Once the app loads:

  1. Open the browser's developer console
  2. Type runMobileTests() and press Enter
  3. Review the test results in the console

What's being tested

  • Viewport configuration
  • Touch target sizes (min 44x44px)
  • Font sizes for readability
  • Horizontal overflow detection
  • Mobile detection
  • Responsive images
  • PWA capabilities
